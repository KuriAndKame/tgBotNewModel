для установки данного модуля требуется edge_tts asyncio watchdog	
данный модуль запускается через edge_tts.py 
данный код берет текстовый файл из папки по пути "D:\Тексты новостей"
названия обработанных новостей записывается в processed_files.txt данный файл отвечает за отсутствие повторных аудио файлов
выгрузка аудио файлов происходит в папку audio_files
изменение выходного пути происходит через переменную output_folder 
изменение выходного пути происходит через переменную input_folder 
изменение пути для файла с сохранеными названиями обработанных файлов происходит через переменную processed_items_path
